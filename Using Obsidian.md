## Linking to other vaults

Use [ ] ( ) syntax with the following or similar as the link:
obsidian://Methods/Statistics/ANOVA
## Tags
Use a hashtag or YAML formatting e.g. 
```yaml
---
tags:
  - recipe
  - cooking
---
```
